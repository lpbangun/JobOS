## Variant: Agent Pair Workspace

### Design stance
Job search as pair work with an agent: conversation is primary, structured JobOS state is a live side panel.

### Key choices
- Layout: split chat / context (OpenCode-like)
- Emphasis: natural-language orchestration over MCP tools
- Interaction: prompt chips, streamed tool call cards, context tabs
- Aesthetic: soft terminal IDE, violet agent accent

### Trade-offs
- Strong at: multi-step delegated work, explaining tradeoffs, mid-flow steering
- Weak at: scanning a large pipeline at a glance; less "board-native"

### Best for
Users who want Hermes/Codex to drive JobOS while they supervise and review drafts.
