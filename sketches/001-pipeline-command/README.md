## Variant: Pipeline Command Center

### Design stance
A keyboard-first ops desk where the kanban pipeline is the product surface, and agents are helpers you summon on a selected card.

### Key choices
- Layout: 3-pane (views + board + selected context)
- Emphasis: application pipeline / stage movement
- Interaction: j/k selection, one-key actions (p pursue, d daily)
- Aesthetic: dark terminal board, dense but still board-readable

### Trade-offs
- Strong at: daily pipeline triage, stage overview, action on selected role
- Weak at: conversational agent collaboration, long research reading

### Best for
Users who think in funnel stages and want a dedicated job-search cockpit.
